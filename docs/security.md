# 安全中心 (Security Center)

Smart Link Manager 采用企业级安全架构，守护每一份点击数据。

## 1. 身份与认证
- **scrypt 哈希**：所有用户密码均经过多轮加盐哈希，防御彩虹表攻击。
- **JWT 强校验**：生产环境强制要求 256 位以上密钥长度。

## 2. 边界防护
- **动态速率限制**：针对每 IP 实施 100 次/分钟的跳转限制，防止流量攻击。
- **系统黑名单**：预置全球 10,000+ 恶意爬虫与扫描器指纹。
- **安全头响应**：强制注入 X-Frame-Options 与 CSP 策略。

## 3. 隐私与合规
- **IP 匿名化**：统计链路仅保留城市信息，不持久化存储原始访问 IP。
- **数据加密**：敏感配置字段（如 AI API Key）采用数据库级加密存储。
