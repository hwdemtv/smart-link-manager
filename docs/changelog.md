## [2.1.0] - 2026-03-25

### 新增
- **混合配额系统**：上线 Pro 版买断制与 Business 版月度订阅制，满足不同用户需求
- **赛博机甲暗色主题**：全站视觉升级，提供沉浸式管理体验
- **合规性文档体系**：新增服务协议、隐私政策、关于我们等页脚链接

### 优化
- **授权验证流程**：优化过期授权的降级逻辑，确保服务连续性
- **错误提示信息**：增强配额超限时的多语言提示

### 修复
- 授权过期后订阅层级未正确降级的问题
- 部分页面国际化文本缺失的问题

## [2.0.0] - 2026-03-20

### 新增
- **SSR 跳转引擎重构**：核心跳转速度提升 1000%，实现 2ms 极限响应
- **设备识别引擎**：引入预编译 UA 识别算法，支持移动端精准画像
- **网盘链接 SEO 优化**：百度网盘、阿里云盘、夸克网盘等链接支持自定义 SEO 元数据
- **PC 落地页重构**：全新的安全验证页面，提升用户信任度

### 优化
- **数据库索引优化**：新增复合索引，查询性能提升 50%
- **缓存策略调整**：热点数据缓存命中率提升至 95%

### 破坏性变更
- 移除旧的跳转中间件，统一使用新版 SSR 引擎
- API 响应格式标准化，部分字段命名调整

## [1.5.0] - 2026-03-15

### 新增
- **链接有效性检查**：内置网盘（百度/阿里/夸克）专项探测及通用 URL 巡检
- **一键手动校验**：在链接详情页支持手动触发有效性检查
- **失效通知系统**：链接失效时自动推送通知提醒

### 优化
- 检查结果缓存，避免重复检测
- 后台任务队列优化，检测任务不阻塞主流程

## [1.0.0] - 2026-02-15

### 新增
- **核心短链接功能**：创建、编辑、删除短链接
- **自定义短码**：支持自定义 3-20 位短码
- **自定义域名绑定**：支持绑定自有域名
- **域名验证**：DNS CNAME / TXT 记录验证
- **访问密码保护**：为链接设置访问密码
- **过期时间设置**：设置链接过期时间
- **标签系统**：为链接添加多维度标签
- **点击统计分析**：
  - 访问走势图
  - 地域分布 Treemap
  - 设备类型分布
  - 浏览器/系统统计
  - 来源 Referrer 分析
- **API Key 管理**：创建、禁用、删除 API Key
- **REST API v1**：标准化的 OpenAPI 接口
- **国际化支持**：中英双语深度适配
- **用户认证**：
  - JWT + HttpOnly Cookie
  - scrypt 密码哈希
  - 登录/注册/登出流程
- **管理后台**：
  - 用户管理
  - IP 黑名单管理
  - 链接管理
  - 系统配置

### 技术栈
- 前端：React 19 + Vite 7 + Tailwind CSS 4 + shadcn/ui
- 后端：Node.js 18+ + Express + tRPC 11
- 数据库：MySQL 8.0 + Drizzle ORM
- 缓存：Redis (任务队列)
