# 隐私政策 (Privacy Policy)

*最后更新日期：2026-03-25*

**Smart Link Manager** 极其重视用户与访客的隐私。本政策说明我们如何收集、使用和保护您的数据。

## 1. 数据的收集
为了提供精准的点击分析服务，当用户访问通过本平台生成的短链接时，我们会收集以下非个人身份标识信息 (Non-PII)：
- **IP 地址**：用于识别大致的地理位置（国家/城市）。
- **设备指纹**：包括操作系统、浏览器版本及设备类型。
- **引用来源 (Referrer)**：了解访客是从哪个页面跳转而来。

## 2. 数据的匿名化处理
- 我们对存储的 IP 地址进行掩码处理，仅保留统计所需的地理位置属性，不存储完整的原始 IP。
- 点击数据仅用于分析报表，不会向任何第三方广告商出售。

## 3. Cookie 的使用
- 我们仅在用户登录管理后台时使用必要的 Session Cookie，以维持账户会话。
- 访客在点击短链接跳转过程中，本平台默认不植入任何追踪类 Cookie。

## 4. 数据安全
- 所有的传输过程均通过 TLS/SSL 加密。
- 系统架构采用了多级防护策略，防止数据库被非法入侵或数据泄露。

## 5. 用户权利
- 用户有权随时删除其生成的链接及相关的统计历史数据。
