# API 文档

Smart Link Manager 提供全栈高可靠的短链接分发能力。

## 1. 核心跳转接口
`GET /:shortCode`
- 将短码映射为目标 URL。
- **性能指标**：重定向开销 < 3ms。
- **功能支持**：单链密码验证、设备定向、过期自动拦截。

## 2. 开发者集成 (tRPC)
我们推荐使用 tRPC 进行深度集成，以获得全栈类型安全支持。
- `links.create`: 生成新的短链接。
- `links.list`: 获取用户下全量链接。
- `stats.getLinkStats`: 获取地理位置、热力图等详细指标。
